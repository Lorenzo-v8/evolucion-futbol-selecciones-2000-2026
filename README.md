# Evolución del rendimiento competitivo en selecciones nacionales

Análisis de datos deportivos que estudia cómo evolucionó el rendimiento de las 
selecciones nacionales de fútbol entre 2000 y 2026, enfocándose en tres ejes:

- ⚽ **Patrones de goles**: variación en la cantidad de goles anotados por partido 
  a lo largo de las temporadas.
- 🏟️ **Ventaja de local**: impacto estadístico de jugar como local frente a jugar 
  como visitante.
- 📈 **Cambios temporales**: tendencias y transformaciones en el estilo de juego 
  y los resultados a lo largo de más de dos décadas.

## Tecnología utilizada
- R

## Fuente de datos
Kaggle: https://www.kaggle.com/datasets/patateriedata/all-international-football-results
